# COVID-19-New-Case-Forecasting-for-Travel-Planning
Forecasted daily COVID-19 new cases for 7 Indian subcontinent countries using WHO data and an ETS (exponential smoothing) model in R. Converted records to time-series with xts(), generated 14-day prediction intervals, and evaluated accuracy with MAPE across four windows (avg 20.25%), noting cycles, high Î±, and key limitations.
